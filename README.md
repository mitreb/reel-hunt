## Setup

1. Install dependencies:

```bash
npm install
```

2. Configure environment:

```bash
cp .env.example .env
```

Add your TMDB API key to `.env`:

```
NUXT_TMDB_API_KEY=your_tmdb_api_key
```

3. Development:

```bash
npm run dev
```

4. Production:

```bash
npm run build
npm run start
```
